# webpack-react-start-kit

## Features
* React, ES6
* JavaScript & TypeScript Support   
    * `*.js`, `*.jsx`
    * `*.ts`, `*.tsx`
* Asset Management  
    * `*.css`, `*.scss`, `*.sass` by bundled into separated files   
    * images files

* Webpack-dev-server with HMR
* Code splitting  


## Usage

### DevServer 
```
$ yarn start 
```

### Build 
```
$ yarn build 
```
